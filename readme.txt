// --- requirements -----------------------------------------------------------------
Install Python:
https://www.python.org/downloads/
Install an IDE for editing and debugging (optional, but recommended):
Visual Studio Code (https://code.visualstudio.com/)
Pycharm (https://www.jetbrains.com/es-es/pycharm/)
...
Install necessary packages.
If using the pip package manager, the commands are:
pip install numpy
pip install matplotlib
pip install opencv-contrib-python
*** EXTRAS **********
If you want to create a virtual environment to avoid overwriting packages and other root configurations:
a) Create a virtual environment:
python -m venv "directory_to_specify"
b) Activate the virtual environment
Run the script: "directory_to_specify/Scripts/activate.bat"
c) Install necessary packages (step 3)
